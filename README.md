Counterpunchers Fan Hub

Overview

Counterpunchers Fan Hub is a community-focused website concept built around podcast discussion, community engagement, topic submissions, episode archiving, events, and source-based research.

Features

* Mobile responsive design
* Discord community hub
* Topic submission board
* Community voting concept
* Receipts and sources library
* Community directory
* Event calendar section
* Episode archive
* Merch showcase section

Design Direction

The visual identity combines:

* Boxing culture
* Streetwear aesthetics
* Independent media
* Community discussion
* Comic-book inspired visuals
* High-energy promotional poster design

Color Palette

Primary

* Red (#d71920)
* Black (#050505)
* White (#f7f2e9)

Secondary

* Yellow (#ffd13b)
* Green (#44b33d)

Site Sections

Hero Section

Primary call-to-action area introducing visitors to the Counterpunchers community.

Discord Hub

Community gathering area for discussions, events, and coordination.

Counterpunch Board

Topic submission and voting system designed to surface future discussion ideas.

Receipts Library

Repository for articles, timelines, documents, research, and supporting sources.

Community Directory

Member showcase highlighting contributors, moderators, researchers, and creators.

Events Calendar

Watch parties, live discussions, meetups, and special events.

Episode Archive

Searchable collection of podcast episodes, clips, notes, and future transcript support.

Merch Wall

Community sayings, branding concepts, and merchandise showcase.

Technical Architecture

Current prototype:

* Single HTML file
* Embedded CSS
* Embedded JavaScript
* Local browser storage
* GitHub Pages compatible
* No backend required

Vision

Create a central home for Counterpunchers community members to discuss ideas, share sources, archive content, coordinate events, and help grow the broader community ecosystem.
